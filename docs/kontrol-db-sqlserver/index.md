# kontrol-db-sqlserver



[jvm]\
The SQL Server module of the KontrolDB database migration tool.



##  Installation



Add a dependency in your `build.gradle.kts` file:

```kotlin
dependencies {
    implementation("net.futureset.kontroldb:kontrol-db-sqlserver:0.1.0")
}
```
